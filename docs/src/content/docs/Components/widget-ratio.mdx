A component that maintains a specific aspect ratio for its child content.

The `WidgetRatio` widget ensures consistent proportional dimensions regardless of screen size. This is particularly useful for media content, cards, and responsive layouts.

- **Responsive**: Maintains proportions across screen sizes
- **Customizable**: Supports any ratio value and border radius
- **Flexible**: Works with any child widget
- **Clipping**: Supports custom clipping behavior

## Properties

<div class="docs-table">
| Property       | Type                    | Description                                                    |
|----------------|-------------------------|----------------------------------------------------------------|
| `ratio`        | `double`                | The aspect ratio to enforce (width / height)                   |
| `child`        | `Widget`                | The widget to size according to the specified aspect ratio     |
| `clipBehavior` | `Clip`                  | How to clip the child content                                  |
| `borderRadius` | `BorderRadiusGeometry?` | The border radius to apply to the widget                       |
</div>

## Usage Guidelines

- Use for content where maintaining specific proportions is important
- Great for media content like images and videos
- Helpful for creating consistent card layouts
- Use `clipBehavior: Clip.antiAlias` when applying rounded corners for smooth edges
- Consider using with `borderRadius` for enhanced visual design

## Accessibility

- Ensure content within the aspect ratio container remains accessible
- Avoid cutting off important information when content is resized
